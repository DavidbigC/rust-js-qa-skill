# rust-js-qa-skill

QA skill package for Rust + JavaScript/TypeScript projects.

## Contents

- `qa-skill.skill`: packaged skill archive
- `qa-skill-unzipped/`: extracted skill files
  - `qa-rust-js/SKILL.md`
  - `qa-rust-js/references/`
  - `qa-skill/` (legacy name variant)

## Purpose

Provides workflows for:

- writing unit tests
- running full regression
- fixing failing tests
- full production-readiness QA

## Install/Use

1. Install the `qa-rust-js` skill into your agent skills directory.
2. Trigger it with `$qa-rust-js`.
